# Lab 1

Try to translate the `light_on.c` code into straight assembly and use the `skel.S` file to compile and flash the MSP430. From there, try to use the two onboard LEDs (red is on P1.0 and green is on P1.6) and toggle them back and forth. A special delaying function has been provided for you in the `skel.S` file that you may need to use. Read the comments!
